* https://cdn.jsdelivr.net/gh/edp1096/hello-prosemirror/dist/myeditor.css

## Build

```powershell
# All
yarn all
# Browser JS
yarn js
# Module
yarn mjs
# CSS
yarn css
```

## Watch

* Browser JS
```powershell
# Browser JS
yarn watch js
# Module
yarn watch
# or
yarn watch mjs
```

## Run Uploader
```powershell
cd dev/uploader
./run.cmd
```
