# Taste ProseMirror

[Page](https://edp1096.github.io/hello-prosemirror) - Require own server for upload testing

## Build

```powershell
# All
yarn all
# Browser JS
yarn js
# Module
yarn mjs
# CSS
yarn css
```

## Watch

* Browser JS
```powershell
# Browser JS
yarn watch js
# Module
yarn watch
# or
yarn watch mjs
```
