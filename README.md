# Taste ProseMirror

[Page](https://edp1096.github.io/hello-prosemirror)

## Build

* All
```powershell
yarn all
```

* Browser JS
```powershell
yarn js
```

* Module
```powershell
yarn mjs
```

* CSS
```powershell
yarn css
```

## Watch

* Browser JS
```powershell
yarn watch js
```

* Module
```powershell
yarn watch
# or
yarn watch mjs
```
